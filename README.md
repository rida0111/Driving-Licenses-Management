# 🚗 DVLD - Driving License Management System

A Windows Forms (C#) application for managing driving licenses.  
This project demonstrates how to build a desktop application connected to a SQL Server database.

---

## 📌 Features
- Manage people and drivers.
- Add, update, and delete licenses.
- Handle applications, renewals, and suspensions.
- Manage users and authentication.
- Integrated with SQL Server database.

---

## 🛠️ Technologies
- C# (WinForms)
- .NET Framework
- SQL Server
- ADO.NET
